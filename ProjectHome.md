.Net Version Checker shows details of all the versions and service packs of .net frameworks installed on your machine.

**System Requirement**

Operating System: Windows XP/Vista (all versions)/Windows 7 (All Versions: and all windows servers.

In case of windows XP, .Net Framework 2.0 or later must be installed in order for application to work.

**Technical Details**

This application is written in C# and .Net Framework 2.0.
